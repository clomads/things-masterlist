## Description


When I got my 97 Ford Aerostar, the dash cubby was really gross and the fabric inside was soaked with *something* - so I designed a replacement. IIRC it isn't perfect, so you may want to make changes, but it was good enough that I kept it in for the 2 years I owned the van.
